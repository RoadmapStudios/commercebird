Media endpoint for WooCommerce API. Upload and list media file by WooCommerce REST API. 

== Description ==
Wooventory is an extension for the WooCommerce API. This plugin allows you to upload product images from app.wooventory.com (free plan) and create purchase orders. 

List all media: available parameters [here](https://developer.wordpress.org/rest-api/reference/media/#list-media)

[wooventory](https://wooventory.com) (WooCommerce Inventory Management app) user: please install this plugin to manage images from wooventory.

== Installation ==
1. Install the Wooventory plugin
2. Activate the plugin
